# 🌌 Heyxi - Link Hub 

Персональная мультиссылочная визитка для **Хэй-Кси**, созданная с использованием современного веб-стека. Проект полностью автономен и работает без внешних зависимостей (Offline Ready).

## ✨ Особенности проекта
*   **🌈 RGB-Design:** Анимированная RGB-рамка карточки и переливающийся текст имени.
*   **🖱️ Interactive UX:** Звуковой отклик при кликах (встроен в JS через Base64).
*   **🌌 Particle Background:** Динамическое звездное небо на фоне с использованием `particles.js`.
*   **📋 Discord Integration:** Функция копирования Discord ID в один клик с эффектом встряски кнопки.
*   **🥚 Easter Egg:** Секретная пасхалка — 10 кликов по аватару активируют вращение и взрыв частиц.
*   **📱 Responsive:** Полная адаптация под мобильные устройства.
